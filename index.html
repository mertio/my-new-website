IBM Research & Development Lab in Germany

Starting from January 2020 until July 2020, I did a voluntary internship at the IBM Research & Development Lab in Germany as a Test Automation Developer. I developed and refactored automated test cases for the IBM StoredIQ Insights project. I tried to make improvements on the existing test cases for the project's user interface. To do this I had to work with the test automation tool Katalon Studio. I used Github as the code repository along with the Zenhub extension, for the organization and documentation of the projects I did throughout the internship. I was also responsible for integrating these automated test cases into the build pipeline, using Travis CI. During this process I used Docker. I also suggested moving the all of the automated test cases from Katalon Studio to Selenium in order to improve the quality, speed, flexibility, and structure of the automated tests. For this I used the Java programming language and the Selenium framework. I also successfully integrated the new Selenium code into the Travis CI build pipeline using Docker containers.

Argela - R&D Company of TURK TELEKOM





Starting from June 2019 until September 2019, I did my mandatory internship at Argela as a Software Developer. During my internship I was busy doing two main tasks: one was to develop a “Call Simulator” for testing one of their products. I developed it from scratch using Spring Boot, Angular and an H2 in-memory database. I implemented log4j for logging and I wrote unit tests for the controllers of the back-end using the JUnit framework. The other task was to learn about SIP (Session Initiation Protocol), a telecommunications protocol, and to research a more different and recently developed server optimized to use SIP in order to run their products in the future. In the end I deployed a Sip Servlet WAR file to a server I found and tested if everything worked well. I also configured its logging mechanism in compliance with the new server and managed to run the server in a Docker container. I reported the details of the process to management.

Software Development Spring Boot Java Angular TypeScript H2 Database Docker log4j JUnit SIP Servlets

Determining the Certain Times Characters Appear in a Simpsons Episode Using Image Recognition with Convolutional Neural Networks



In this project the goal will be to use convolutional neural networks and deep learning to build an application that takes in a Simpsons episode and a character as input and returns in what parts of the episode the character appears as output. The use of this application will be making compilation videos of Simpsons characters a lot easier. Instead of scanning through hours of footage in the Simpsons archive to find the scenes of the character of interest, it will now be possible to just choose the character and get what time frame they appear in in the episode.
